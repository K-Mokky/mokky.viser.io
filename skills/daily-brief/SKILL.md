# Daily Brief
Description: Turn scattered notes, reminders, and recent context into a concise daily plan.

## When to use
Use this when the user asks for a daily briefing, priority list, morning plan, or end-of-day review.

## Procedure
1. Summarize known commitments and unresolved tasks.
2. Identify the top three priorities.
3. Flag blockers, deadlines, and messages that need replies.
4. Return a short actionable plan with time blocks when enough timing context exists.
